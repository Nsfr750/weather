# Weather App

An enhanced weather application that provides current weather information and forecasts with a graphical user interface. The app features location detection, weather visualization, and detailed weather metrics.

## Features

- Current weather information
- Weather forecasting with visualizations
- Automatic location detection
- Temperature, wind speed, and humidity plotting
- User-friendly GUI interface

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Nsfr750/weather.git
   ```

2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Get an API key from [OpenWeatherMap](https://openweathermap.org/api) and set it in the app's settings menu.

## Usage

1. Run the application:
   ```bash
   python weather.py
   ```

2. Select a city or use automatic location detection
3. View current weather and forecast data
4. Access additional features through the menu bar

## Requirements

- Python 3.x
- OpenWeatherMap API key
- Internet connection for weather data retrieval
